# SPARC RPG Design Assets

## Source Files
- `../Sparc-StyleGuide 2.pdf` - Brand guidelines, colors, typography
- `../Sparc UI comps.pdf` - Full UI mockups and component designs

## Extracted Assets

### Logo (`../` root)
- `sparc-logo.png` - Main SPARC logo (golden text on light background)

### Style Guide Images (`./` this folder)
- `logo-002.png` - Main logo with icon
- `logo-003.png` - Logo white/glow variant
- Various other brand elements

### UI Component Extracts (`./ui-comps/`)
- Individual UI elements extracted from the UI comps PDF
- Larger files (10MB+) are full page mockups
- Smaller files are individual components

### Full Page Renders (`./ui-pages/`)
- Full page PNG renders of each PDF page at 150dpi
- Best for reviewing complete screen designs

## Key Design Elements (from Style Guide)

### Colors (approximate)
- Primary Orange: #CC7A00
- Bronze: #8B4513  
- Gold: #FFB347
- Dark Background: #121212

### Typography
- See Style Guide PDF for font specifications

## For PRD Authors
When writing design-related PRDs, reference:
1. The style guide PDF for brand guidelines
2. The UI comps PDF for screen designs
3. Extracted assets in this folder for specific elements
